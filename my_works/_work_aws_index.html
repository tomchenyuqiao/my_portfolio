Project 2

Project 3


For my last project in this course, I leveraged AWS technologies to process "real time" finance data. I configured a Kinesis Delivery Stream along with a S3 bucket to place finance data. Next, I wrote a lambda function to drop data onto the Kinesis stream. Finally, a crawler from AWS Glue was configured to create a table to query in Athena.
